Diseño: Neumorfismo Cyberpunk + Glassmorphism

Estructura táctil con transparencias difuminadas y acentos de neón

Previsualización de Componentes

Tarjeta de Perfil

Este componente usa sombras duales para simular volumen físico sobre una superficie oscura.

Efecto Cristal (Glass)

Ideal para modales, sidebars o elementos flotantes.

Propuesta Técnica Detallada

  • Atmósfera: Profundidad absoluta mediante el uso de sombras suaves en superficies oscuras (Neumorfismo oscuro).
  • Capas de Cristal: Elementos secundarios (como el fondo del Navbar o modales) usarán el efecto backdrop-filter: blur() para un toque moderno.
  • Acentos Reactivos: Neones en Cyan y Magenta que se iluminan al interactuar, contrastando con la base mate.
  • Habilidades Vanilla: Todo el efecto de "volumen" y "cristal" se logrará mediante CSS puro (sombras, gradientes y filtros), demostrando que no se necesita Tailwind ni librerías externas para un acabado de alta gama.
